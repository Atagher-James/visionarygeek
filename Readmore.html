Hi! I'm Atagher James and I am learning frontend development, and this is my first project. In web development, we have two sections of design: the first is frontend development which is basically the client side/users of the web, and it is built with HTML, CSS, and Java script. HTML stands for Hyper-text markup language, and it provides the structure of the website. CSS stands for cascading style sheet; as you may guess from the name, yes! it is used for styling. CSS helps you to design the structure provided by HTML. Javascript is used for defining the functionality of the website. Javascript makes the website look more lively as you could have a lot of animations that are peculiar to using power point for presentation. Another section of the web design is the Back end which connotes the Server section of the web design.

Talking about the back end which is the most important tool of the programmer, it allows the programmer to decide what content to give to the users and how to give it. This section is only important to the owner of the website. It combines languages other than HTML AND CSS. We use Javascript, PHP, and even python for this design. PHP is used to handle forms and anything of such nature, while Python is used for data structure.

I built this site with only HTML AND CSS, no wordpress, no template, it is pure coding. I haven't mastered this two language so this site may not look all that professional, besides this is my first project. Once I begin to learn Javascript I'm sure of producing something more appealing to the eye. I would love to see your rating and suggestion on this site. Thanks for reading.
